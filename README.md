# Moodle Activity Module: Recall

Recall is a modern, single-page application (SPA) flashcard activity plugin for Moodle that utilizes the proven Leitner system of spaced repetition.

## Description

The Recall activity allows students to practice flashcards using a spaced repetition system based on the Leitner method. It provides a modern, fast, and responsive user interface built as a Vue.js 3 single-page application integrated directly into Moodle.

Teachers can easily manage flashards manually or import them in bulk using AI-generated text. The plugin tracks student progress, supports custom activity completion rules based on cards mastered, and helps students retain knowledge efficiently.

## Features

- **Leitner System:** Cards are organized into boxes based on difficulty and review results. 
- **Modern UI:** Built with Vue.js 3 and Tailwind CSS for a seamless, page-reload-free experience.
- **AI Bulk Import:** Teachers can paste a simple text format generated by ChatGPT/Claude to instantly create dozens of cards.
- **Activity Completion:** Set requirements based on minimum cards reviewed or minimum cards perfectly mastered (reaching the final box).
- **Responsive:** Works perfectly on desktop and mobile devices.

## Requirements

- Moodle 4.1 or later.
- No additional PHP extensions required.

## Installation

1. Downoad the latest release `.zip` file.
2. Log in to your Moodle site as an administrator.
3. Go to **Site administration** > **Plugins** > **Install plugins**.
4. Upload the zip file and follow the onscreen instructions.
5. Alternatively, you can extract the zip file directly into the `mod/recall` directory of your Moodle installation and trigger the upgrade via the browser or CLI.
